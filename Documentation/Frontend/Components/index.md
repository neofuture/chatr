# Components

All reusable UI components are located in `frontend/src/components/`. Each component lives in its own folder with an `index.md` reference document.

## Structure

```
components/
├── messaging/
│   ├── MessageBubble/
│   ├── MessageAudioPlayer/

│   ├── Select/
│   ├── Textarea/
│   ├── Checkbox/
│   ├── Radio/
│   ├── RangeSlider/

│   ├── DatePicker/

│   ├── BottomSheet/

│   └── Lightbox/
├── image-manip/

├── forms/
│   ├── LoginForm/
```

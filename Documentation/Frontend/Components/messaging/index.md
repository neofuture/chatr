# Messaging Components

Components responsible for rendering conversations, composing messages, and playing chat media.

| Component | Description |
|-----------|-------------|
| [ConversationsList](./ConversationsList/index.md) | Scrollable user list with search, presence, last message, and flip animation |
| [ConversationView](./ConversationView/index.md) | Combined ChatView + MessageInput panel — the full chat experience |
| [ChatView](./ChatView/index.md) | Scrollable message list with typing/recording indicators and reactions |
| [MessageInput](./MessageInput/index.md) | Full input bar — text, emoji, file attachments, voice recording, reply/edit mode |
| [MessageBubble](./MessageBubble/index.md) | Renders individual message bubbles with grouping, status, and type handling |
| [MessageAudioPlayer](./MessageAudioPlayer/index.md) | Audio player with waveform visualisation and listening status |
| [EmojiPicker](./EmojiPicker/index.md) | Full emoji picker with categories, search, and recently used panel |
| [VoiceRecorder](./VoiceRecorder/index.md) | Microphone recording modal with live waveform capture |

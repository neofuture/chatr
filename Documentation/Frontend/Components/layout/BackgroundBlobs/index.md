# BackgroundBlobs

**File:** `src/components/BackgroundBlobs/BackgroundBlobs.tsx`

Renders three decorative animated blobs used on the landing page background. No props.

## Props

None.

## Usage

```tsx
<BackgroundBlobs />
```

The blobs are styled and animated purely via CSS classes `bg-blob-1`, `bg-blob-2`, `bg-blob-3` defined in the global stylesheet.

# Panels

Stacked sliding panel system for modal-like navigation flows. Panels animate in from the right (or left), can be nested up to any depth, and support custom title bars with avatars, subtitles, and action buttons.

| Component | Description |
|-----------|-------------|
| [PanelContainer](./PanelContainer/index.md) | Renders the active panel stack — mount once in layout |
| [AuthPanel](./AuthPanel/index.md) | Combined login/register sliding panel |
| [DemoPanels](./DemoPanels/index.md) | Interactive demo of all panel configuration options |

See also: [`PanelContext`](../../Contexts/index.md) for the `usePanels` hook API.
